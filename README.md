# Udemy Clone

This project is a simple clone of the Udemy landing page built using **HTML and CSS**.  
The main purpose of this project is to practice **Flexbox layout**, website structuring, and basic frontend development skills.

## 🚀 Features

- Navigation bar similar to Udemy
- Course category section
- Course cards layout
- Flexbox based responsive layout
- Clean and simple UI design

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox

## 📚 What I Learned

Through this project I learned:

- How to structure a webpage using HTML
- How to align elements using Flexbox
- Creating reusable layout sections
- 
- Building a real-world UI clone
